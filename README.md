Условие:
Базовый класс: геометрическая фигура (содержит координаты центра фигуры, абстрактрные методы, 
возвращающие координаты прямоугольника, в который заключена фигура, площадь фигуры).
Производные классы: точка, линия, многоугольник, эллипс. У линии и точки площадь равна 0, 
у остальных - вычисляются. У каждой фигуры задаются свои свойства, определяющие её геометрию.

GeometricFigure - родительский клас для остальных классов Figure
Классы Point, Line, Ellipse, Rectangle - классы-наследники класса GeometricFigure,
в каждов классе реализованы характерные для каждой фигуры методы.
